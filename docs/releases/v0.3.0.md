# v0.3.0

Status: release candidate until the release gate is fully verified.

## Added

- `app`, `api`, and `brochure` response profiles.
- Conservative profile auto-detection with confidence and decision evidence.
- Explicit `--profile` override.
- Profile-aware applicability, findings, partial credit, and 100-point scoring.
- Version-pinned citation catalogue with primary specifications, OWASP ASVS 5.0.0,
  official guidance, NIST control-informed mappings, and peer-reviewed CSP research.
- Self-contained, responsive, print-aware HTML report with no JavaScript or remote
  runtime assets.
- Default URL query and fragment redaction with explicit `--include-query` override.
- Cross-origin redirect blocking with an explicit authorized-scope override.
- Narrow `GET` fallback only after `HEAD` returns `405` or `501`.
- Deterministic JSON fixtures for application, API, brochure, and hostile evidence.
- Privacy, accessibility, authorization, methodology, and release-gate documentation.

## Changed

- Replaced the universal baseline score with profile-specific weights.
- Recognize CSP `frame-ancestors` as the modern framing control when
  `X-Frame-Options` is absent.
- Expanded Markdown and JSON output with profile decision and methodology data.

## Security And Privacy

- Escapes untrusted values in HTML output.
- Neutralizes raw HTML and structural injection in Markdown output.
- Rejects non-HTTP schemes and credentials embedded in target URLs.
- Redacts query and fragment data by default.
- Prevents redirect-driven origin expansion by default.
- Keeps reports local and dependency-free.
